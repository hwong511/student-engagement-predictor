# Data Directory

## Data Use Agreement

This directory contains data under **Data Use Agreement (DUA)** with Carnegie Learning.

**Data files are NOT committed to version control** and cannot be shared publicly due to privacy and proprietary restrictions.

---

## Required Data Files

To run this project, place the following files in this directory:

### Input Files (Required)

**1. BROMP Behavioral Observations**
- **File:** `BROMPclean.csv`
- **Description:** Human-coded behavioral observations using BROMP protocol
- **Size:** ~500 KB
- **Rows:** 4,037 observations
- **Source:** Live classroom observations across 13 classes

**2. Caliper Digital Event Logs**
- **File:** `caliperclean.csv`
- **Description:** IMS Caliper-formatted digital learning events
- **Size:** ~2.5 MB
- **Rows:** 15,068 events
- **Source:** Carnegie Learning MATHStream platform

### Generated Files (Created by Scripts)

**3. Merged Features Dataset**
- **File:** `prediction_features.csv`
- **Description:** Merged BROMP and Caliper data with engineered features
- **Generated by:** `scripts/01_merge_datasets.R`
- **Size:** ~800 KB
- **Rows:** 4,037 (one per BROMP observation)
- **Columns:** 40+ features

**4. Full Merged Dataset**
- **File:** `+-90sec_merged_data.csv`
- **Description:** Complete merged dataset with symmetric time windows
- **Generated by:** `scripts/01_merge_datasets.R`
- **Contains:** All temporal features (backward, forward, symmetric)

**5. Stream-Normalized Data** *(Optional)*
- **File:** `caliper_processed_data.csv`
- **Description:** Event logs normalized by learning stream
- **Generated by:** `scripts/02_exploratory_stream_analysis.R`

## Privacy & Ethics

### Data De-identification

All data has been de-identified following best practices:

- Student IDs replaced with anonymous codes
- No personally identifiable information (PII)
- Classroom identifiers anonymized
- Temporal patterns preserved for analysis
- Geographic information removed

### Ethical Use

This data is used solely for:

- Educational research purposes
- Methodological development
- Academic portfolio demonstration

**Not used for:**
- Commercial purposes
- Individual student evaluation
- Public release or sharing
